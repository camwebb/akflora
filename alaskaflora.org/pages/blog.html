Development Blog

This website presents step-by-step developments in software, data, and documentation as we progress towards the end goal of a comprehensive informatics platform for the new Flora of Alaska. We will “live-blog” the process to share the decisions, discoveries (and mistakes!) so as to help anyone working on a similar proplem.

Please note: this website and all related resources for the project exist as a Github repo.
